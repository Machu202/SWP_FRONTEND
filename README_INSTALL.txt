STATIC HTML/CSS/JS SRC PACKAGE

This package changes your frontend from React src/ into a static HTML/CSS/JS src/ folder.

How to install in your project:
1. Extract this zip.
2. Copy the src folder into:
   C:\Users\admin\Documents\GitHub\SWP_FRONTEND
   Replace the existing React src folder.
3. Copy package.json and vite.config.js into:
   C:\Users\admin\Documents\GitHub\SWP_FRONTEND
   Replace the old ones if needed.
4. In VS Code terminal, run:
   npm install
   npm run dev
5. Open:
   http://localhost:5173

Main pages:
- index.html
- dashboard.html
- tantou-review.html
- editorial-board.html

Core flow:
index.html
→ dashboard.html
→ tantou-review.html
→ editorial-board.html
