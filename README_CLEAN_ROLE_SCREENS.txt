CLEAN ROLE SCREENS PATCH

This patch removes the old Mangaka and Assistant screens and replaces them with new clean role-based pages.

Install:
1. Stop Vite with Ctrl + C.
2. Delete your old src folder.
3. Copy this src folder into your SWP_FRONTEND project.
4. Replace package.json and vite.config.js with the included versions.
5. Run:
   npm install
   npm run dev
6. Open:
   http://localhost:5173/

Main role entry pages:
- index.html
- mangaka-dashboard.html
- assistant-dashboard.html
- tantou-dashboard.html
- board-dashboard.html
- admin-dashboard.html

Old Mangaka/Assistant page names such as dashboard.html, series.html, assignments.html, and page-editor.html are intentionally not included.
