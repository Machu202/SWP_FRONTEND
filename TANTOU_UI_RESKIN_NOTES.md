# Tantou UI Reskin Patch

This patch updates only the **visual presentation** of the Tantou Editor screens.

## What changed
- Reskinned these pages to match the provided editorial dashboard references:
  - `src/tantou-dashboard.html`
  - `src/tantou-review.html`
  - `src/tantou-feedback.html`
  - `src/tantou-revision.html`
  - `src/tantou-report.html`
- Added a new Tantou visual theme in `src/style-workspace.css`.

## What did NOT change
- Existing routes
- Existing API calls
- Existing `role-integration.js` backend logic
- Existing working features and state flow
- Admin / Editorial Board functionality

## Integration
Replace the corresponding files in your current frontend package, or use the full reskinned export zip.
